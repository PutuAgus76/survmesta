Selamat datang di SURVMESTA! Ini adalah server Survival murni di mana pemain dapat berpetualang, membangun, dan bertahan hidup bersama pemain lain. Server ini dimainkan secara santai namun tetap seru.

1. CARA LOGIN (Penting!)

Saat pertama kali masuk server, pemain wajib mendaftarkan akun agar aman.

Perintah:

/register <password> <ulangi_password>

Contoh:

/register rahasia123 rahasia123

Jika sudah pernah daftar, gunakan:

/login <password>

2. MENUJU DUNIA SURVIVAL

  • Saat join, pemain akan berada di Lobby.
  • Cari NPC di depan pemain.
  • Klik NPC untuk teleport ke dunia Survival (SURVMESTA).
  • Untuk kembali ke Lobby gunakan perintah:
/lobby

3. FITUR TIMBER (Tebang Pohon Instan)

  • Fitur ini membuat pohon hancur seketika saat ditebang.
  • Disarankan dimatikan saat membangun rumah dari kayu.

Perintah:

  • Menyalakan: /on timber
  • Mematikan: /off timber

Status ON/OFF dapat dilihat di scoreboard sebelah kanan layar.

4. SISTEM SCOREBOARD

Scoreboard di sebelah kanan layar menampilkan:

  • Nama
  • Uang ($)
  • Status Timber
  • Ping

Perintah untuk menampilkan atau menyembunyikan:

/sb

5. TELEPORTASI (TPA)

Gunakan perintah berikut untuk berkunjung ke teman:

  • /tpa <nama_teman>
  • /tpaccept
    (Untuk menerima)
  • /tpdeny
    (Untuk menolak)

6. PANDUAN SHOP (/shop)

Gunakan perintah /shop untuk membeli dan menjual item.

üì± KHUSUS PENGGUNA HP (BEDROCK / MCPE)

A. Cara Membeli Barang:

  • Geser item ke inventory untuk membeli.
  • Gunakan ikon Glass (1, 16, 32) untuk mengatur jumlah.
  • Gunakan ikon Chest untuk membeli 1 stack (64 item).

B. Cara Menjual Barang:

  • Masuk kategori barang.
  • Geser ikon Torch (Obor) untuk masuk Mode JUAL.
  • Geser item dari inventory ke menu shop untuk menjual otomatis.
